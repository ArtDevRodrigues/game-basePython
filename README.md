# Game-basePython

# Sobre o Projeto

Projeto usando Python que simula um game de rodadas, nele cada um dos players ataca por vez, tem chance de um player atacar mais de uma vez.

# Tecnologias usadas

-Python 3.+

# Pré-Requisitos
## As versões mais recentes de:
- Python 3.+
